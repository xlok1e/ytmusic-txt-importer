# YTMusic Importer

Переносит треки из текстового списка в плейлист YouTube Music.

---

## Скачать и запустить

Перейди в раздел [**Releases**](../../releases) и скачай файл для своей системы.

### macOS

1. Скачай `YTMusic.Importer.macOS.zip`
2. Разархивируй — появится `YTMusic Importer.app`
3. Дважды кликни на приложение

> **Если macOS заблокировала запуск:**  
> Кликни правой кнопкой на `.app` → **Открыть** → в диалоге снова нажми **Открыть**.  
> Это нужно сделать только один раз.

### Windows

1. Скачай `YTMusic Importer.exe`
2. Дважды кликни на файл

> **Если Windows Defender заблокировал:**  
> Нажми **"Подробнее"** → **"Выполнить в любом случае"**.

После запуска автоматически откроется браузер с интерфейсом.  
Не закрывай чёрное окно терминала — это работающий сервер.

## Как пользоваться

### Шаг 1 — Подготовь список треков

Создай `.txt` файл, в котором каждая строка — один трек.  
Формат: `Исполнитель — Название` или просто `Название`. Например:

```
Arctic Monkeys — Do I Wanna Know?
Radiohead — Creep
The Strokes — Reptilia
```

### Шаг 2 — Получи заголовки запроса из YouTube Music

Это нужно для авторизации — приложение будет действовать от твоего имени.

1. Открой [music.youtube.com](https://music.youtube.com) и войди в аккаунт
2. Нажми `F12` (или `Cmd+Option+I` на Mac) — откроются инструменты разработчика
3. Перейди на вкладку **Network** (Сеть)
4. Поставь лайк на любой песне — в списке запросов появится `like`
5. Кликни на этот запрос правой кнопкой → **Copy** → **Copy request headers**
6. Убедись, что в скопированном тексте есть строка `Authorization: SAPISIDHASH ...` — без неё ничего не работает

### Шаг 3 — Импортируй

1. Вставь скопированные заголовки в поле **Request Headers**
2. Напиши название будущего плейлиста
3. Загрузи `.txt` файл с треками (перетащи или нажми на область)
4. Нажми **Начать импорт**

Прогресс отображается в реальном времени. Если остановить импорт — приложение запомнит прогресс и можно продолжить позже.

## Сборка из исходников

<details>
<summary>Инструкция для разработчиков</summary>

### Требования

- Python 3.9+

### Установка и запуск

```bash
pip install -r requirements.txt
python launcher.py
```

### Сборка бинарника

```bash
pip install pyinstaller
pyinstaller ytmusic_importer.spec
```

- **macOS** → `dist/YTMusic Importer.app`
- **Windows** → `dist/YTMusic Importer.exe`

> `.exe` собирается только на Windows, `.app` — только на macOS.  
> GitHub Actions собирает оба автоматически при создании нового Release.

</details>
